Developed tools




TRACESPipe is an automatic and efficient pipeline for identification, assembly, and analysis of viral genomes and human-host genomes that combines the DNA sequence data from multiple organs. TRACESPipe relies on cooperation among 3 modalities: compression-based prediction, sequence alignment, and de novo assembly. The pipeline is ultra-fast and provides, additionally, secure transmission and storage of sensitive data.

Code: TRACESPipe

Public Data


  • B19V.fa: assembled B19V genome using TRACESPipe.
  • JCPyV.fa: assembled JCPyV genome using TRACESPipe.
  • HMT.fa: assembled human mitogenome (host) using TRACESPipe.
